เกี่ยวกับฉัน

ฉันชื่อนางสาววันวิสา โลหะชาติ มีความสนใจด้าน Web Application, การพัฒนาเว็บไซต์ และฐานข้อมูล

สามารถทำงานกับ HTML, CSS, JavaScript, SQL, API และ Web Service ได้

ทักษะทางเทคนิค

1. ภาษาโปรแกรม: Dart, HTML, CSS, PHP ใช้พัฒนาแอปและเว็บไซต์

2. ฐานข้อมูล: MySQL, SQL (คำสั่ง SELECT, JOIN, INSERT, UPDATE)

3. Web Development: WordPress, REST API, JSON

4. Framework / Library: Vue.js (พื้นฐาน), React (พื้นฐาน)

5. เครื่องมือและอื่น ๆ: Git, Visual Studio Code, Postman

6. ระบบปฏิบัติการ: Windows, Linux

ความสามารถ

1. การพัฒนาเว็บไซต์มีทักษะการสร้างเว็บไซต์ด้วย WordPress เช่น การพัฒนาและจัดการเว็บไซต์ การใช้งานปลั๊กอิน การออกแบบ UI/UX การจัดการคอนเทนต์ และการรักษาความปลอดภัย

2. การออกแบบ UI/UX : ใช้เครื่องมือ Figma, Canva หรืออื่นๆ สำหรับออกแบบประสบการณ์ผู้ใช้และอินเทอร์เฟซ

3. การใช้เครื่องมือพัฒนา : ใช้งาน Visual Studio Code และมีพื้นฐาน Flutter สำหรับสร้างแอป Android และ Web

4. การใช้งาน Microsoft 365 : ใช้งาน Word, Excel, PowerPoint ในระดับพื้นฐานเพื่อการทำงานเอกสารทั่วไป
